HOTEL CONTROL TOWER ENTERPRISE
==============================

1. Install Python 3.11 or 3.12.
2. Extract this folder.
3. Double-click run_app.bat.

Manual commands:
  python -m pip install -r requirements.txt
  python -m streamlit run Hotel_Control_Tower_Enterprise.py

Demo accounts:
  admin / admin123
  manager / manager123
  reception / front123
  restaurant / pos123
  kitchen / kitchen123
  housekeeping / house123
  accounts / accounts123

The SQLite database hotel_erp.db is created automatically in this folder.
Keep approvals.py in the same folder as the main application.
